■ Notepad++ Autohotkey 개발환경 만들기 / DDART, 2020.11.13, http://auto.ddart.net
===============================================================
1. 코딩시 자동완성 기능 추가 (AutoHotkey.xml 파일 복사) 
  가. C\AHK\UTIL\notepad++_autohotkey\autoCompletion\AutoHotkey.xml  위 파일을 
       C:\Program Files (x86)\Notepad++  autoCompletion폴더 생성후 복사
  나. 노트패드++에서 : 설정 → 환경설정 → 자동완성 
      ▣ 각 입력에 자동 완성 사용 : 체크 -  함수 및 단어 완성 체크 확인
      ▣ 입력시 함수 인자 정보표시 : 체크

2. 사용자 정의언어 추가 (C:\AHK\UTIL\notepad++_autohotkey\userDefinedLang_AHK.xml)
  - 노트패드++ : 언어 → 사용자 정의 언어 → 언어 정의 → 가져오기(userDefinedLang_AHK.xml)

3. NppExec 플러그인 설치 및 설정
  플러그인 → 플러그인 관리 → 사용가능 → 찾기에서 NppExec 검색후 설치
  플러그인 → NppExec → Show NppExec Console → 콘솔 아웃풋 윈도우 보기
  플러그인 → NppExec → Console Output...{OEM/OEM} 에서 Output Input 모두 UTF-8로 변경
  → 콘솔 출력시 한글등 유니코드 문자깨짐 방지.

4. 실행키 설정 : 2가지 방법
 가.  플러그인 → NppExec → 실행 → Command 입력창에
      "C:\Program Files\AutoHotkey\AutoHotKey.exe" "$(FULL_CURRENT_PATH)" 입력(""포함)후 저장(Save)
       → F6단축키로 소스편집중 바로 실행가능. Ctrl+F6으로 이전명령 바로 실행

 나. F5키(실행할 프로그램) → $(FULL_CURRENT_PATH) 입력 → 저장(단축키 지정) 사용

  ※ 추후 ahk 파일은 Notepad++ 에서 열어 인코딩 → UTF-8 BOM으로 변경저장 한번씩 필요

기타 :  functionList parser 추가(2개파일 복사) ※ 생략 가능
   C:\AHK\UTIL\notepad++_autohotkey\autoCompletion 폴더 안에 
   2개 파일(ahk.xml, overrideMap.xml)을 C:\Users\사용자\AppData\Roaming\Notepad++\functionList 에 복사
   보기 → 함수 목록 → 함수, 클래스 목록뷰 제공으로 소스 편집, 분석, 탐색이 수월
